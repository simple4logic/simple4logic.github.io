basic git commmand
==============================

깃헙 업로드를 위한 간단한 명령어 정리
<br>
<br>
***

1. **git bash** 실행
---
\simple4logic.github.io 로 이동 후 git bash 실행   
<br>
<br>
***

2. 명령어 입력
----

```
git status
```
>파일 변경 상태를 알려주는 명령어 
<br>

```
git add .
```
>어떤 파일을 올릴지 결정하는? 명령어
<br>

```
git commit -m "commit한 내용"
```
> commit해서 staging area로 올리고, 어떤 commit을 했는지 설명하는 명령어
<br>

```
git push
```
> 실제로 깃헙 서버에 올리는 명령어
